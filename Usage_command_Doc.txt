================================================================================
                           QUERY SYNTAX TIPS
================================================================================

1. Use quotes for exact field names: 'customer_id'
2. All filters are case-insensitive
3. Use "under" or "in" or "from" interchangeably
4. Use "with more than N" for numeric filters
5. Use "containing" for keyword search
6. Use "matching pattern" with * for wildcards
7. Combine multiple filters with "where", "and", commas
